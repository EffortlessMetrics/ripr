# Allowlisted non-Rust programming and declarative implementation surfaces.
#
# Format:
# glob|kind|owner|reason
#
# Rust is the default implementation language for repo automation, production
# logic, test harnesses, fixture runners, release checks, and policy checks.
# New non-Rust programming files must match one of these entries or add a new
# reviewed entry with an owner and reason.

*.md|documentation|docs|Root documentation.
.cargo/config.toml|cargo_config|rust|Cargo aliases for repository tooling.
.github/ISSUE_TEMPLATE/*.yaml|github_template|repo-process|GitHub issue forms.
.github/ISSUE_TEMPLATE/*.yml|github_template|repo-process|GitHub issue forms.
.github/dependabot.yml|github_dependency_automation|security|Dependabot update configuration for Cargo, npm, and GitHub Actions.
.github/workflows/*.yaml|ci_declarative|release/ci|GitHub Actions workflow definitions.
.github/workflows/*.yml|ci_declarative|release/ci|GitHub Actions workflow definitions.
.gitignore|repo_config|repo|Git ignore rules.
.ripr/*.toml|policy_metadata|repo-policy|Traceability metadata.
.ripr/*.txt|policy_metadata|repo-policy|Policy allowlists.
.ripr/goals/*.toml|policy_metadata|repo-policy|Codex Goals campaign metadata.
AGENTS.md|documentation|agents|Agent instructions.
CHANGELOG.md|documentation|docs|Repository changelog.
CONTRIBUTING.md|documentation|docs|Contribution guide.
