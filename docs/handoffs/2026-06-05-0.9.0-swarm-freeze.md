Date: 2026-06-05

## Why this is a freeze note

This handoff records the release-candidate state of `ripr-swarm` for the
next source release. From this point, normal feature work pauses: only
release blockers land before the source sync.

## Version decision

`0.9.0`, not `0.8.1`.

- `0.8.0` was published from source `ripr` on 2026-06-02 (crates.io
  `0.8.0`, tag `v0.8.0`, source sync merge `e5e4766a` / PR #1422).
- The post-0.8.0 delta (`74b1fab0..1cce26df`, 100 commits) contains
  user-visible TypeScript/Bun adapter behavior and Perl preview-card
  surfaces, which under the project versioning rule is a minor-release
  event, not a patch.
- Naming discrepancy: several planning docs and handoffs refer to the
  TypeScript/Bun preview slice as "0.8.1" (e.g.
  `2026-06-05-0.8.1-typescript-bun-preview-patch-proof.md`,
  `2026-06-05-post-081-typescript-bun-support-decision.md`). That slice
  was an internal planning name and was never published (crates.io
  `max_version` is `0.8.0`); the work ships in `0.9.0`. Do not rewrite
  the old documents.
