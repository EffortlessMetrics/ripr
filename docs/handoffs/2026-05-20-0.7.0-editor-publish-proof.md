## Badge Boundary

Marketplace badge refresh was intentionally not included in this publish proof.

The repo policy requires static VS Marketplace install-count badge updates to
come from publisher metrics or to be explicitly left unchanged. This release
proof did not open the publisher metrics page and did not update public badge
counts. Any badge endpoint or static install-count refresh remains a separate
scoped badge PR.

Open VSX live Shields badges may continue to resolve from Open VSX, but no
badge files were edited by this release proof.
