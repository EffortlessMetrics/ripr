## Claim boundary

This packet supports the narrow patch claim:

```text
For calibrated Bun stable-byte cases, ripr can provide advisory
cross-language TypeScript preview evidence that distinguishes TS-discriminated,
missing-discriminator, mention-only, bridge-unknown, and named-limitation
states while keeping public repair packets disabled.
```

It does not support claims that:

- TypeScript or JavaScript preview is stable or Rust-parity analysis.
- The Bun binding graph is complete.
- Runtime Bun, Jest, Vitest, `tsc`, `tsserver`, Miri, provider calls, or
  mutation execution ran.
- `ripr` generated tests, edited Bun source, or proved UB behavior.
- Default CI gates, badges, baselines, RIPR Zero, support tiers, release
  records, publishing, signing, marketplace artifacts, or install docs changed.
